# Python Learning Repository

A comprehensive collection of Python learning materials covering fundamental concepts, object-oriented programming (OOPs), and practical applications.

## 📁 Repository Structure

### Core Python Concepts
- **Beginner.py** - Introduction to Python basics
- **if__name==main.py** - Understanding `__name__` and main execution context
- **Global and Local Variable.py** - Scope and variable management
- **IS and ==.py** - Comparison operators and identity vs equality
- **ShortHand_if_else.py** - Ternary operators and conditional expressions
- **lambda.py** - Anonymous functions and functional programming
- **decorator.py** - Decorator patterns and function wrapping
- **map filter.py** - Functional programming with map and filter functions
- **Enumarate.py** - Using enumerate for iteration with index
- **Importing.py** - Module and package imports

### Object-Oriented Programming (OOPs)
Located in the **OOPs/** folder:
- **into oops.py** - Introduction to OOP principles
- **Class and objet.py** - Creating and using classes
- **constructor.py** - Constructors and initialization
- **inheritence.py** - Inheritance and class hierarchy
- **Access Modifier.py** - Public, private, and protected members
- **getter and setter.py** - Property accessors and mutators
- **static meathod.py** - Static methods and class methods
- **ClassMethod.py** - Class methods and their usage
- **super keyword.py** - Using super() for parent class access
- **Alternative Constructors.py** - Factory methods and alternative instantiation
- **instence var VS class var.py** - Instance vs class variables
- **__dict__,help.dir.py** - Introspection tools and object inspection

### File Input/Output
Located in the **file IO/** folder:
- **main.py** - Main file I/O operations example
- **readline.py** - Reading file line by line
- **Seek AND Tell.py** - File pointer manipulation
- **myfile.txt** - Sample text file for I/O operations

### Operating System Module
Located in the **OsModule/** folder:
- **Create.py** - Creating files and directories
- **Delete.py** - Removing files and directories
- **rename.py** - Renaming files and directories
- **Oslist.py** - Listing directory contents

### Miscellaneous Topics

- **raising_Custom_error.py** - Custom exceptions and error handling
- **finallyKeword.py** - Try-except-finally blocks
- **Library manage.py** - Managing Python libraries and dependencies

## 🚀 How to Use

1. **Start with basics**: Begin with `Beginner.py` to understand fundamental concepts
2. **Progress systematically**: Move through files in order of complexity
3. **Explore OOPs**: Navigate to the **OOPs/** folder to learn object-oriented concepts
4. **Practice file operations**: Use **file IO/** folder examples for file handling
5. **System operations**: Check **OsModule/** for operating system interactions

## 💡 Tips

- Execute any `.py` file to see the code in action
- Modify and experiment with the code to deepen understanding
- Refer to examples when implementing your own projects
- Use comments within each file to understand logic
